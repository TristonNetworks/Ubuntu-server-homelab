# Ubuntu-server-homelab
Linux server administration and networking labs using Ubuntu Server while studying for CompTIA A+ and Network+.

This repository documents my Linux server administration labs using Ubuntu Server.

These labs are part of my preparation for:

- CompTIA A+
- CompTIA Network+

## Lab Topics

1 Installation of Ubuntu Server
2 Linux User Management
3 DHCP Server Configuration
4 DNS Server Configuration
5 NAT Router Configuration
6 Network Troubleshooting
7 System Monitoring

## Tools Used

Ubuntu Server
Virtual Machines
Wireshark
Linux CLI

## Goal

Develop hands-on Linux and networking skills through practical labs.
